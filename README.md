## ðŸš¦ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/ProjectTusharKumar/live.git
cd live
```

### 2. Install Dependencies

```bash
npm install
# or
yarn install
```

### 3. Run the Development Server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
